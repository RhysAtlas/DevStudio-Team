# DevStudio Team

Projeto simples de site institucional desenvolvido para a disciplina de Front-End.

A ideia foi criar uma landing page (one page) apresentando uma equipe fictícia, seus serviços e alguns projetos, usando apenas HTML, CSS e JavaScript.

---

## 📄 Sobre

O site simula a página de uma equipe de desenvolvimento, com seções básicas que normalmente aparecem em sites reais:

* Home (apresentação)
* Quem somos
* Serviços
* Projetos
* Contato

Tudo foi feito em uma única página com navegação por scroll.

## ⚙️ O que foi implementado

* Estrutura de página institucional
* Layout com CSS (flexbox)
* Seções organizadas em one page
* Scroll suave entre seções
* Formulário de contato (apenas visual)
* Interações básicas com JavaScript

---

## 🔧 Organização do projeto

A gente dividiu o trabalho por área:

* HTML → estrutura da página
* CSS → layout e estilos
* JS → interações

Também usamos GitHub para versionamento:

* Cada um trabalhou em uma branch separada
* As mudanças foram unidas via Pull Request
* `dev` para integração
* `main` com a versão final
